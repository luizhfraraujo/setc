event-details Luiz Henrique de Freitas Rodrigues Araújo Consumindo API Rest com Angular/Ionic

Criação de uma API REST simples utilizando NodeJS como backend, armazenando os dados em um banco de dados NoSQL (Mongo). Em seguida, vamos criar um aplicativo mobile utilizando o framework Ionic, tendo como base o Angular, que irá acessar e exibir as informações da nossa API criada. O palestrante é Arquiteto de Sistemas na empresa LinkedBy e entusiasta de novas tecnologias. Trabalho hoje dando manutenção e desenvolvendo novas integrações no sistema de Business Intelligence, sistema este que atende em torno de 400 empresas do ramo de bebidas e distribuição, como grupo Heineken, Solar (Coca Cola), Grupo Petrópolis (Itaipava).
